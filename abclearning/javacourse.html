JAVA

Java is a programming language that produces software for multiple platforms. When a programmer writes a Java application, the compiled code (known as bytecode) runs on most operating systems (OS), including Windows, Linux and Mac OS. Java derives much of its syntax from the C and C++ programming languages. The basic steps to create the Hello World program are: write the program in Java, compile the source code, and run the program.

  1. Write the Java Source Code
  2. Save the File
  3. Open a Terminal Window
  4. The Java Compiler
  5. Change the Directory
  6. Compile Your Program
  7. Run the Program
